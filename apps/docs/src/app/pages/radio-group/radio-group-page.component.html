Radio Group

A set of mutually exclusive options. Supports vertical and horizontal orientations, three sizes, signal binding, and full keyboard navigation.

ngAtoms v0.3+ ngatoms add radio-group Stable Source

Usage

Add the radio group to your project and import both the group component and the radio directive.

$ ngatoms add radio-group

Then import into your component:

Vertical (default)

Radios stack vertically by default. Bind [(value)] for two-way signal binding. The selected value is shown below.

Selected: {{ plan() }}

Horizontal

Set orientation="horizontal" for inline option groups.

Sizes

Three sizes. Set size on individual ngAtomsRadio inputs.

sm
md
lg

Disabled

Disable individual radios with the native disabled attribute.

API

Inputs on nga-radio-group. The ngAtomsRadio directive takes value and size.
