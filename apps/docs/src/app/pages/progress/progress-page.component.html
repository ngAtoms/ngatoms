Progress

Displays a value within a known range as a filled bar, or an indeterminate animation when the completion percentage is unknown.

ngAtoms v0.3+ ngatoms add progress Stable Source

Usage

Add the progress component to your project and import the standalone component wherever you need it.

$ ngatoms add progress

Then import into your component:

Variants

Four semantic variants. default uses the foreground color, accent uses the brand accent, success signals completion, and destructive signals a critical state.
