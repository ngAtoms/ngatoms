Badge

Small inline label for status, counts, and categories. Seven semantic variants keep feedback visually distinct without extra structure.

ngAtoms v0.3+ ngatoms add badge Stable Source
New Active Pending Deprecated

Usage

Add the badge to your project and import the standalone directive wherever you need it.

$ ngatoms add badge

Then import into your component:

Variants

Seven visual styles. Use default for general labels, semantic variants (success, warning, destructive) for statuses, solid for high emphasis, and outline for subtle labeling.

@for (variant of variants; track variant) { {{ variant }} }

With icons

Any content — text, icons, or both — is allowed inside the badge. Use Phosphor Icons for consistent sizing.

Stable Beta Deprecated Angular

As link

Apply ngAtomsBadge to an anchor tag when the badge should navigate. It keeps the badge appearance but is a real link.
