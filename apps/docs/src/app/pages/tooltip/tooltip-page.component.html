Tooltip

A floating label that appears on hover or keyboard focus. Appended to document.body via position: fixed for reliable stacking at any nesting depth.

ngAtoms v0.3+ ngatoms add tooltip Stable Source

Usage

Add the tooltip to your project and apply it as an attribute directive.

$ ngatoms add tooltip

Import NgAtomsTooltipDirective in your component:

Placements

The tooltip auto-positions to the specified side. top is the default.

On different elements

Apply ngAtomsTooltip to any element — buttons, badges, spans, or custom components.

Beta

Long content

Tooltips wrap at max-width: 16.25rem by default. Use them for brief contextual descriptions, not for multi-paragraph content — use a Popover for that.
