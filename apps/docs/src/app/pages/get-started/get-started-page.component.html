ngAtoms

Get started.

Angular UI primitives you own. Install a CLI, copy components into your project, and ship — no runtime dependency, full control.

~ 2 min Angular 17+ Node 18+ ngatoms.com
1

Install the CLI

Install globally once. Use npx ngatoms if you prefer not to install globally.

2

Initialize your project

Run once at the root of your Angular workspace. It asks three short questions and sets up a config file and an optional design token theme.

init never overwrites existing files — if a conflict is detected, it prompts before touching anything.
3

Add a component

Components are copied into your src/components/ directory. From that point on they are yours — edit, delete, rename, fork.

You can add multiple components at once: ngatoms add button input select
4

Use it in a template

Import the standalone directive and add it to your component's imports array. No module, no provider, no extra setup.

5

You're done.

No build steps to configure. No runtime wrapper. No version mismatches. The source lives in your repo — open it, read it, adapt it.

Theming

Every color, radius, and font is a CSS variable in src/themes/default.css, generated by ngatoms init. Edit it freely — the whole system reflows.

Dark mode is a .dark class on <html> or <body>. Toggle it with one line: document.documentElement.classList.toggle('dark')
