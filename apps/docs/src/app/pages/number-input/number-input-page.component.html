Number Input

A numeric input with increment and decrement controls, min/max clamping, and step support. Bound to a signal — the display stays in sync as the value changes.

ngAtoms v0.3+ ngatoms add number-input Stable Source

Usage

Add the number input to your project and import the standalone component wherever you need it.

$ ngatoms add number-input

Then import into your component:

Sizes

Three sizes. md is the default and works in most form layouts.

sm
md
lg

Min, max, and step

Clamp the value with [min] and [max]. Change the increment amount with [step] — supports decimal values.

Min 1 / Max 10
Step 0.5

Disabled

Pass [disabled]="true" to prevent interaction.
