Components / Forms

Switch

A toggle control for enabling or disabling a setting. Three sizes, signal binding, and keyboard-accessible — drop it on any <input> element.

ngAtoms v0.3+ ngatoms add switch Stable Source

Usage

Add the switch to your project and import the standalone directive wherever you need it.

$ ngatoms add switch

Then import into your component:

States

Bind [(checked)] for two-way signal binding. Use the native disabled attribute to prevent interaction.

Sizes

Three sizes. md is the default and suits most form layouts.

Settings list

A common pattern — row with label on the left and switch on the right, spanning the full container width.

Push notifications Receive alerts for new activity
Auto-save Automatically save changes as you work
Analytics Share usage data to help improve the product

API

Inputs on ngAtomsSwitch. The native disabled attribute passes through to the underlying element.
