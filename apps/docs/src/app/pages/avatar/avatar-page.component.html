Avatar

Displays a user image with graceful fallback to initials or a generic icon. Four sizes and automatic broken-image detection.

ngAtoms v0.3+ ngatoms add avatar Stable Source
@for (size of sizes; track size) { }

Usage

Add the avatar to your project and import the standalone component wherever you need it.

$ ngatoms add avatar

Then import into your component:

Image

Provide a src URL and an alt description. The image fills the avatar circle and is cropped to cover.

@for (size of sizes; track size) {
{{ size }}
}

Initials fallback

Set fallback to a short string (typically 1–2 characters) shown when no image is provided or the image fails to load.

@for (size of sizes; track size) {
{{ size }}
}

Icon fallback

When neither src nor fallback is set, a generic user icon is shown as the final fallback.

@for (size of sizes; track size) {
{{ size }}
}

Broken image

If the image URL fails to load, the component automatically falls back to initials (if set) or the generic icon. No extra configuration needed.
