Select

A custom dropdown with optional search, multiple selection, keyboard navigation, and three visual variants. Fully accessible and composable.

ngAtoms v0.3+ ngatoms add select Stable Source

Usage

Add the select to your project, then import the component and option type.

$ ngatoms add select

Import in your component:

Variants

Three visual weights. default for standard form fields, filled for surfaces that already have a background, ghost for minimal inline usage.

Sizes

Five sizes. md is the default. Use sm / xs in dense toolbars and lg / xl in prominent form areas.

Searchable

Pass [searchable]="true" to show a search field at the top of the panel. Options filter as the user types.

Multiple selection

Pass [multiple]="true" to allow selecting more than one option. The value binding becomes string[].

@if (skills().length > 0) { Selected: {{ skills().join(', ') }} }

States

Invalid state is for form validation errors. Disabled prevents any interaction.
