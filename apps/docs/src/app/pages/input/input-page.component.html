Components / Forms

Input

Text input field with three visual variants, five sizes, invalid state, and full native attribute pass-through. Drop it on any <input> element.

ngAtoms v0.3+ ngatoms add input Stable Source

Usage

Add the input to your project and import the standalone directive wherever you need it.

$ ngatoms add input

Then import into your component:

Variants

Three visual styles. default shows a clear border, filled uses a subtle background without a border, and ghost is borderless for inline editing contexts.

@for (variant of variants; track variant) {
{{ variant }}
}

Sizes

Five sizes from xs to xl. md is the default and fits most form layouts.

@for (size of sizes; track size) {
{{ size }}
}

States

Use [invalid]="true" for validation errors. Native disabled and readonly attributes pass through to the underlying element.

Default
Invalid
Disabled
Readonly

API

Inputs on ngAtomsInput. All standard input attributes pass through to the underlying element.

Prop Type Default Description
@for (prop of props; track prop.name) {
{{ prop.name }} {{ prop.type }} {{ prop.defaultVal }} {{ prop.description }}
}

Accessibility

Built on a native <input>. Associate a <label> or use aria-label for screen readers.
