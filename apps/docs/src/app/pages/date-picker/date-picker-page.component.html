Date Picker

A fully custom calendar overlay with single and range selection, min/max/available-date constraints, month and year pickers, and i18n support.

ngAtoms v0.3+ ngatoms add date-picker Stable Source

Usage

Add the date picker to your project and bind the value via a signal.

$ ngatoms add date-picker

Import in your component:

Single date

The default mode. Bind a single ISO date string with [(value)].

@if (single()) { {{ single() }} }

Range selection

Set mode="range" to enable a start/end date selection. Bind both with [(startDate)] and [(endDate)].

@if (rangeStart()) { {{ rangeStart() }} → {{ rangeEnd() || '...' }} }

Constraints

Limit the selectable dates with minDate, maxDate, or an availableDates allowlist.

Min / Max {{ minDate }} → {{ maxDate }}
Available dates Only 5 specific dates enabled

Sizes

Three trigger sizes: sm, md (default), and lg.

Internationalization

Override locale, dayLabels, and monthLabels to localise the calendar for any language.
