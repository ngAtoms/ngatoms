Accordion

A vertically stacked set of interactive headings that reveal or hide associated sections. Supports single and multi-panel modes with full keyboard navigation.

Usage

Add the accordion to your project and import all four standalone pieces wherever you need them.

$ ngatoms add accordion

Then import into your component:

Single (default)

Only one panel is open at a time. Opening a new panel collapses the previously open one. Set collapsible="false" to prevent closing the last open panel.

Multiple

Set type="multiple" to allow any number of panels open simultaneously. Useful for FAQs or settings groups where users need to compare across sections.
