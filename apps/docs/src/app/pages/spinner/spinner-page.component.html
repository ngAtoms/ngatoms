Spinner

A circular loading indicator for async operations. Inherits color from its CSS context and comes in three sizes.

ngAtoms v0.3+ ngatoms add spinner Stable Source

Usage

Add the spinner to your project and import the standalone directive wherever you need it.

$ ngatoms add spinner

Then import into your component:

Sizes

Three sizes. md is the default and works inline with body text.

@for (size of sizes; track size) {
{{ size }}
}

Colors

The spinner inherits the current CSS color property. Set it on the element or a parent to change the spinner color.

Foreground
Accent

API

Inputs on ngAtomsSpinner. Apply to any inline element — typically a <span>.

Prop Type Default Description
@for (prop of props; track prop.name) {
{{ prop.name }} {{ prop.type }} {{ prop.defaultVal }} {{ prop.description }}
}

Accessibility

Includes role="status" and a default aria-label="Loading" for screen readers.
