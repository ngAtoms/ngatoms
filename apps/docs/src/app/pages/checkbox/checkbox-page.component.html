Checkbox

A control that allows the user to toggle between checked, unchecked, and indeterminate states. Three sizes, full signal binding, and keyboard-accessible.

ngAtoms v0.3+ ngatoms add checkbox Stable Source

Usage

Add the checkbox to your project and import the standalone directive wherever you need it.

$ ngatoms add checkbox

Then import into your component:

States

Bind [(checked)] for two-way signal binding. Use [indeterminate]="true" for partial selection, and the native disabled attribute to prevent interaction.

Sizes

Three sizes. md is the default for most form fields. Use sm for dense layouts and lg for touch targets or prominent settings.

In a list

Wrap each checkbox in a <label> to extend the tap target to the full row.

API

Inputs on ngAtomsCheckbox. The native disabled attribute passes through to the underlying input.
