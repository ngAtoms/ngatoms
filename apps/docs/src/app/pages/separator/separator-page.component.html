Separator

A visual divider between content sections. Supports horizontal (default) and vertical orientations with full ARIA semantics.

ngAtoms v0.3+ ngatoms add separator Stable Source

Usage

Add the separator to your project and import the standalone directive wherever you need it.

$ ngatoms add separator

Then import into your component:

Horizontal

The default orientation. Stretches to the full width of its container. Use to separate stacked content sections.

Vertical

Set orientation="vertical" for inline use. The parent must be a flex or grid row container so the separator has a defined height.
